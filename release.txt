suppose you are preparing to release 1.12....

create the based-on-1.12 branch:

hg branch based-on-1.12
make changes...
hg commit

when you are finally ready to release 1.12:

hg update default
hg merge based-on-1.12
hg commit -m "merging based-on-1.12"
hg update based-on-1.12
hg update --clean && hg commit --close-branch -m "close branch"
hg update default
grep 1.11 */pom.xml
