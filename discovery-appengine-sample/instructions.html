Instructions for the Google Discovery API App Engine Sample

Browse Online

  • Browse Source, or main file DiscoveryAppEngineSample.java

Checkout Instructions

Prerequisites: install Java 6 and Mercurial. You may need to set your JAVA_HOME.

cd [someDirectory]
hg clone https://samples.google-api-java-client.googlecode.com/hg/ google-api-java-client-samples
cd google-api-java-client-samples/discovery-appengine-sample

Setup Project in Eclipse 3.5/3.6

Prerequisites: install Eclipse, and the Mercurial plugin.

  • Import discovery-appengine-sample project
    • File > Import...
    • Select "General > Existing Project into Workspace" and click "Next"
    • Click "Browse" next to "Select root directory", find [someDirectory]/google-api-java-client-samples/discovery-appengine-sample and click "Next"
    • Click "Finish"
  • Run
    • Right-click on project discovery-appengine-sample
    • Run As > Web Application
