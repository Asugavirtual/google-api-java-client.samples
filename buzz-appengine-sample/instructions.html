Instructions for the Google Buzz API App OAuth 2 Engine Sample

Browse Source Code

  • buzz-appengine-sample or SampleApp.java

Checkout Instructions

Prerequisites: install Java 6, Google App Engine 1.4.0, Mercurial, and Maven. You may need to set your JAVA_HOME.

cd [someDirectory]
hg clone https://samples.google-api-java-client.googlecode.com/hg/ google-api-java-client-samples
cd google-api-java-client-samples/shared/shared-sample-common
edit src/main/java/com/google/api/services/samples/shared/oauth2/OAuth2ClientCredentials.java
mvn install
cd ../../buzz-appengine-sample
mvn clean package

Setup Project in Eclipse 3.5/3.6

Prerequisites: install Eclipse, Google Plugin for Eclipse, and the Mercurial plugin.

  • Setup Eclipse Preferences
    • Window > Preferences... (or on Mac, Eclipse > Preferences...)
    • Select Maven
      • check on "Download Artifact Sources"
      • check on "Download Artifact JavaDoc"
  • Import buzz-appengine-sample project
    • File > Import...
    • Select "General > Existing Project into Workspace" and click "Next"
    • Click "Browse" next to "Select root directory", find [someDirectory]/google-api-java-client-samples/buzz-appengine-sample and click "Next"
    • Click "Finish"
  • Run
    • Right-click on project buzz-appengine-sample
    • Run As > Web Application
