Instructions for the Google Prediction API v1.3 JSON ClientLogin Sample

Browse Online

  • Browse Source, or main file PredictionSample.java

Checkout Instructions

Prerequisites: install Java 6, Mercurial and Maven. You may need to set your JAVA_HOME.

Important: after checking out the project, and before compiling and running it, you need to edit ClientLoginCredentials.java to enter a username and password that has access to the Google Prediction API. Otherwise, you will get a 403 Forbidden exception. You will also need to upload the training data to Google Storage and edit the OBJECT_PATH.

cd [someDirectory]
hg clone https://samples.google-api-java-client.googlecode.com/hg/ google-api-java-client-samples
cd google-api-java-client-samples/cmdline-sample-shared
edit src/main/java/com/google/api/services/samples/shared/oauth2/OAuth2ClientCredentials.java
mvn install
cd ../prediction-cmdline-sample
mvn -q compile exec:java

Setup Project in Eclipse 3.5/3.6

Prerequisites: install Eclipse, the Mercurial plugin, and the Maven plugin.

  • Setup Eclipse Preferences
    • Window > Preferences... (or on Mac, Eclipse > Preferences...)
    • Select Maven
      • check on "Download Artifact Sources"
      • check on "Download Artifact JavaDoc"
  • Import prediction-cmdline-sample project
    • File > Import...
    • Select "General > Existing Project into Workspace" and click "Next"
    • Click "Browse" next to "Select root directory", find [someDirectory]/google-api-java-client-samples/prediction-cmdline-sample and click "Next"
    • Click "Finish"
  • Run
    • Right-click on project prediction-cmdline-sample
    • Run As > Java Application
    • If asked, type "PredictionSample" and click OK
    • To enable logging:
      • Run > Run Configurations...
      • Click on "Java Application > PredictionSample"
      • Click on "Arguments" tab
      • VM arguments: -Djava.util.logging.config.file=${project_loc}/logging.properties
      • Click "Run"
